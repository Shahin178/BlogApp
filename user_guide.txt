# Blog Application Documentation

## Overview
This documentation provides a guide on how to use the Blog Application, a web application developed using the MERN (MongoDB, Express.js, React.js, Node.js) stack. The application is deployed on Render (backend) and Netlify (frontend). It includes features such as user authentication (signup, login, logout), data validation, viewing all blogs, viewing specific blogs, creating, updating, deleting blog posts, and searching blogs by name.

## Features
1. **User Authentication**: 
   - **Signup**: New users can register for an account.
   - **Login**: Registered users can log in using their credentials.
   - **Logout**: Users can log out from their accounts.
2. **Data Validation**: Input data is validated for correct formats and required fields.
3. **View Blogs**: 
   - **All Blogs**: Users can view a list of all available blogs.
   - **Specific Blog**: Users can click on a blog to view its details.
4. **Create Blog**: Users can create new blog posts with a title, image URL, published date, read time, and multiple subheadings.
5. **Update Blog**: Users can update the content of an existing blog post.
6. **Delete Blog**: Users can delete an existing blog post.
7. **Search Blog**: Users can search for blogs based on the blog name.

## Using the Application

### 1. **Sign Up**
- Navigate to the sign-up page.
- Enter the required details (name, email, and password).
- Submit the form to create a new account.

### 2. **Login**
- Navigate to the login page.
- Enter your registered email and password.
- Click "Login" to access your account.

### 3. **Home Page**
- After logging in, you will be redirected to the home page.
- Here, you can view a list of all blog posts.

### 4. **View a Specific Blog**
- Click on any blog card to view the full content of that blog.

### 5. **Search Blogs**
- Use the search bar at the top to search for blogs by their name.

### 6. **Create a New Blog**
- Click on the "Create New Post" button.
- Fill in the details for your new blog post, including the title, image URL, published date, read time, and subheadings.
- Click "Submit" to publish the new blog post.

### 7. **Update a Blog**
- Navigate to the blog you want to update.
- Click the "Update" button.
- Edit the fields you want to change.
- Click "Save" to apply the updates.

### 8. **Delete a Blog**
- Navigate to the blog you want to delete.
- Click the "Delete" button.
- Confirm the deletion in the popup overlay by clicking "Yes".

### 9. **Logout**
- Click on the "Logout" button in the header to log out of your account.
